
# 🎬 Flutter Movie App - Version 1.1

## 🛠️ Technologies & Architecture  

- **Flutter** for Desktop & Web  
- **Native Flutter State Management**: `ValueNotifier` & `ListenableBuilder`  
- **Custom Theming with Light & Dark Modes**  
- **Reusable Custom UI Components**:  
  - `filled_btn_custom.dart`  
  - `outlined_btn_custom.dart`  
  - `switch_decoration.dart`  
  - `textfied_decoration.dart`  

---

## 🎨 Styling Focus  

- 🎨 Global `AppColors` for consistent color usage  
- ✍️ `TextStyleDefault` for typography  
- 🌓 ThemeData structured for **Light & Dark themes**  

---

## 📦 Getting Started  

1. Clone the repository  
2. Run `flutter pub get`  
3. Execute with:  
   ```bash
   flutter run -d chrome
   ```  
   or run on your desktop device  

💡 **Recommended for developers wanting to explore Flutter Web/Desktop differences.**
